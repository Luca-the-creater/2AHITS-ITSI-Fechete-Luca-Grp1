Mit per Pipe `|` verknüpften Befehlen ergeben sich viele Möglichkeiten. Zum Beispiel das Tool sort mit dem Daten sortiert werden können.

----

### Beispiel

Sortiere die Einkaufsliste shopping.txt (download der Datei mit curl), gib dann nur die letzten 5 Zeilen aus.

Die letzten 5 Zeilen der unsortierten Datei:

```sh
$ cat shopping.txt | tail -n5
```
Den Inhalt der Datei in sortierter Reihenfolge:

```sh
$ cat shopping.txt | sort
```

Die letzten 5 Zeilen der sortierten Datei:

```sh
$ cat shopping.txt | sort | tail -n5
```

## Übung (Numeric Sort)
Recherchiere die sort Option -n, --numeric-sort und erkläre (auch anhand eines Beispiels) die Auswirkung wenn diese Option angegeben oder nicht angegeben ist.

Hinweis: Sortiere eine Datei die ein- und zweistellige Zahlen gemischt untereinander enthält.

## Lösung:

Die Option `-n` oder `--numeric-sort`
Normalerweise sortiert die Shell wie im Wörterbuch (alphabetisch). Das ist bei Zahlen aber ein Problem, mit der Option `-n` sagst du der Shell dass sie die Zahlen nach ihrem echten Wert sortieren soll (1, 2, 3...) und nicht nur nach der ersten Ziffer.

z.b.:

```sh
File Zahlen.txt:
1
2
4
6
98
7
97
45
100
3
9182
```
```sh
~/workspace$ sort -n Zahlen.txt
1
2
3
4
6
7
45
97
98
100
9182
```
