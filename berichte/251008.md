# Grundlagen

- Optionen, shorthand, longhand
- Argumente
- `echo`
- History
- Tastenkürzel (strg gehtst du am anfang und mit strg e gehst du am ende und strg c endet das program)
- Filesystem
  - `ls`, `cd`, `pwd`
  - Arbeitsverzeichnis, working directory
  - Unterschied relativer Pfad/absoluter Pfad
    - `mkdir`
    - `rm`
    - `rmdir` das löst alles (rm -r / löst ALLES)


__Erklärung:__

_Optionen aus einem Buchstaben = shorthand_

longhand mit 2x `--`:

```sh
date --iso-8601
```

Mehr Informationen zu einem Befehl:

-KI
-häufig gibt es die Option `--help`

```sh
$ date --help
```

-man pages

```sh
$ man date
```

_Befehl auf Argumente_

befehl = `echo`

```sh
echo Hallo 2AHITS
```
und Ausgabe ist 

```sh
Hallo 2AHITS
```

_Zugriff auf alle Befehle_

befehl = `history`

```sh
history
```

und die Ausgabe ist

```sh
1  echo Hallo Weld
2  echo Hallo 2AHITS Gruppe 1
3  echo -n Test
4  echo test
5  ls
6  cd
7  cd /workspace
8  ...
```
_Tasten trickst_

am Amfang zu gehen

```sh
STRG + a
```

um am Ende zu gehen 

```sh
STRG + E
```

und damit Sie den Progam enden 

```sh
STRG + C
```


_Filesystem_

befehl `ls` zeigt dir von ein Ordner die Daten unt Unterordner

```sh
cd berichte
ls
```
und die Ausgabe ist

```sh
250910.md  251008.md  252809.md  _Fechete_Luca.md  img
```

mit den befhel `cd` steigst du ihn ein Ordner rein

```sh
~/workspace$ cd berichte
```

und die Ausgabe ist

```sh
~/workspace/berichte$ 
```

mit `cd ..` steigt man von den Ordner heraus

```sh
~/workspace/berichte$ cd ..
```
und die Ausgabe ist

```sh
~/workspace$ 
```

mit den befehl `pwd` sieth man die komplete Adresse wo der Pfaden sich befindet

```sh
pwd
```

und die Ausgabe ist

```sh
/home/runner/workspace
```
