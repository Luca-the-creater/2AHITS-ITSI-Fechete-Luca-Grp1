**__Erklärung__**

__Der `cat` befehl__

Die Funktion von `cat` ist es, den Inhalt einer oder mehrerer Dateien auf der Standardausgabe (`stdout`) anzuzeigen.

Mit den Befehl:

```sh
cat >test.txt
```
Schreibt man in den File z.B.

```sh
~/workspace/stdinout$ cat >test.txt
xyz

```

und mit den Befehl:
```sh
cat >test.txt
```

Siet man was in ein File steht
z.B. 
```sh
~/workspace/stdinout$ cat <test.txt 
xyz
```

Man kann mit den Befehl:
```sh
cat hallo.txt test.txt
```
mehrere inhalte aus ein File anzeigen z.B.
```sh
~/workspace/stdinout$ cat test.txt hallo.txt 
Text
Hallo Weld
```
**__ACHTUNG!!! wen man was drinnen in den File hat und man will was schreiben braucht man ein anderer Befehl weil sonst überschreibt man es einfach__**

Der Befehl lautet:

```sh
echo Hallo Weld >>hello.txt
```
die Ausgabe wäre:

```sh 
~/workspace/stdinout$ echo Servus Weld >>hello.txt 
~/workspace/stdinout$ cat hello.txt 
Hallo Welt
Servus Weld
```

In der shell ist `#` ein kommentar


es gibt zwei davon `Quotes` und `single Quotes`


Mit `'` kan man auch abstände machen wie z.B.:

```sh
mkdir 'Urlaub Fotos'
```

ausgabe:

```sh
~/workspace$ mkdir 'Urlaub Fotos'
~/workspace$ ls
 abcd   berichte   new.txt   nocheins   README.md  'SEW Bericht.txt'   stdinout   test.txt  'Urlaub Fotos'
~/workspace$
```

Die Regel gilt auch bei Zeichen z.B. bei `>>> Achtung <<<`
